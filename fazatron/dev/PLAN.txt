Содержание проекта.

0.	Список сокращений и обозначений. Нормативные ссылки.

1.	Постановка задачи. Вместо введения. 
Разработать программу для вычисления синуса угла. Учесть начальные условия: 
	диапазон представления, 
	параметры машины, 
	ошибки констант,
	допустимая область применения(удовлетворительная ошибка на выходе,ограничение по набору комманд) [1]

2.	Разбиваем задачу на компоненты:
		разработка алгоритма
		определение ошибок вычислений
		определение параметров вычислителя
		разработка машинной схемы
		разработка программы

3.	Раздел 1. 
	Разработка алгоритма:
		теоретическое обоснование решаемой задачи:
			метод разложения функции в ряд,
			обоснование выбора данного метода,
			построение блок-схемы алгоритма.

4.	Раздел 2. 
	Определение ошибок вычислений:
		определяемые параметры [2]:
			Исходя из поставленной задачи (минимум машинных ресурсов на решение задачи с заданной точночтью) необходимо
			определить такие свойства среды (например разрядность, количество членов разложения), которые непосредственно 
			влияют на результат работы программы. Их корректировка должна приводить к изменению точности результата, что 
			в свою очередь позволяет решить задачу.
		теория:
			ошибки, содержашиеся в исходной информации,
			ошибки численных методов решения задачи,
			ошибки, возникающие в результате необходимости представлять число в виде конечной последовательности(округление),
			{ошибки, вносимые аппаратурой - в первом приближении константы, а в решаемой задаче - отсутсвуют.},
			влияние ошибок друг на друга и их воздейстие на результат последовательности операций [самарин стр 66,3]
		практика:
			определение ошибок исходной информации, числ. методов,
			исходя из алгоритма - определение ошибок по [3] и ошибок округления,
			запись системы уравнений для ошибок и разрешение ее относительно определяемых параметров. [4]

5.	Раздел 3. 
	Определение параметров вычислителя:
		исходя из [4] и [1] выбирается разрядность вычислителя и количество членов разложения в ряд (и т.д.),
		исходя из алгоритма определяется необхдимый набор базовых комманд с учетом [4],
		выбор методов адресации и работы с памятью.

6.	Раздел 4. 
	Разработка машинной схемы:
		на основании [Раздел 3], с использованием алгоритма [Раздел 1] создается машинная схема,
		которая описывает весь процесс распределения переменных/констант в памяти и последовательность выполнения 
		базовых операций.

7.	Раздел 5.
		На основании машинной схемы [Раздел 4] разрабатывается программа на псевдоязыке с использованием синтаксиса из [Раздел 3].

8.	Заключение. Выводы

9.	Приложение А.
	Метод определения коэффициентов членов разложения функции в ряд по полиномам Чебышева.

10. Приложение Б.
	Методы учета ошибок.

11. Библиографический список.
